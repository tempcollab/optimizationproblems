## Result
`d_B <= 63`. A **321-point** set `X ⊂ R^63` such that every strictly-smaller-diameter subset has `<= 5` points. Hence any partition into smaller-diameter parts needs `>= ceil(321/5) = 65 > 64 = 63+1`. **Margin is exactly 1 part** (65 vs 64 required) — the tightest of all the SRG counterexamples.

## Construction (full chain)
Built on the standard `G_2(4)` Euclidean representation used by Bondarenko / Jenrich–Brouwer.
- **Finite model.** `F_16 = F_2[α]/(α^4+α+1)`, projective plane `PG(2,16)` with Hermitian form `h(u,w)=u0 w0^4 + u1 w1^4 + u2 w2^4`. Vertices = unordered triples of pairwise-orthogonal non-isotropic projective points (the 416 orthogonal bases). Two vertices adjacent iff their associated isotropic 15-sets meet in exactly 3 points. Graph `Γ = G_2(4)`, **srg(416,100,36,20)**, eigenvalues 100/20/−4 (mult 1/65/350).
- **Standard representation (Lemma 2).** Gram matrix `G = 96 I + 24 A − 6 J` is PSD of rank 65 (eigvals 0,576,0 on the three eigenspaces). Gives vectors `x_v ∈ R^65` with `x_u·x_v = 90 / 18 / −6` for `=/∼/≁`, so squared distances `144` (adjacent) / `192` (non-adjacent). Two-distance set.
- **Partition (Lemma 1, verified facts).** Fix first isotropic point `q0`; `B` = vertices containing a non-isotropic point orthogonal to `q0` (`|B|=96`), `C = V\B` (`|C|=320`). Induced graph on `B` has **three components `B1,B2,B3` of size 32**. Degree data: internal `B_i` degree 20, cross `B_i–B_j` = 0, `B_i–C` = 80; for `c∈C`: `|N(c)∩B_i| = 8`, `|N(c)∩C| = 76`.
- **Codimension-2 subspace (Lemma 3).** `S_i = Σ_{y∈B_i} x_y`. Since `x_c·S_i = 8·18 + 24·(−6) = 0`, every `x_c ⟂ S_1,S_2,S_3`. Gram of `(S_1,S_2,S_3)` has eigenvalues `0,18432,18432` ⇒ `dim span = 2`. So `W = span(S1,S2,S3)^⟂` has `dim W = 65−2 = 63`, and all `x_c ∈ W`. **(Re-verified numerically: rank of Gram on C is exactly 63.)**
- **The added point (Lemmas 4–5).** Pick `b∈B1`, `z_b = x_b − (1/32)S_1 ∈ W`, with `‖z_b‖^2 = 78` and `z_b·x_c = 18` (b∼c) / `−6` (b≁c). Set `t = sqrt((222−1)/13)` (positive root of `78t^2+12t=102`), `p = t z_b`. Then `X = {x_c : c∈C} ∪ {p}` ⊂ `W ≃ R^63`, `|X|=321`. Squared diameter 192: `‖p−x_c‖^2 = 192` if `b≁c`, `= 192−48t < 192` if `b∼c`.
- **Clique bound (Lemma 6).** Two `x_c,x_c'` are at full diameter iff `c ≁ c'`. So a smaller-diameter subset ⇒ vertices pairwise **adjacent** ⇒ a **clique** in `Γ`; and `ω(Γ)=5`. Including `p`: `{b}∪{c : x_c∈Y}` is a clique, again `<=5`. Hence every smaller-diameter part has `<=5` points.
